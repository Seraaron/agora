# Specialties List

#imp The following speciealties, uses, and passives are just suggestions; please don't feel the need to look them up every time you use them, or to write them on your sheet exactly as they're written here.

###### A-B

| Acute sense of...                                                                                                                    |
| ------------------------------------------------------------------------------------------------------------------------------------ |
| **Use when:** The given sense is being used predominantly or intensively for an action.                                              |
| _Passively:_ Your FC may choose to inform you of more things relating to that sense than they ordinarily might.                      |
| _**Example senses:** Hearing, Sight, Smell, Taste, Touch, etc, or less common 'senses' like Balance, Empathy, Heat, Direction, etc._ |

| Almanacs                                                                                                                                                          |
| ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Knowledge of calendars, dates, or seasons is important to your action, or use to accurately predict the day's weather (ie. as an established fact). |
| _Passively:_ You always know the approximate time of day and the month of the year, so long as you can see the sun, the moon, or the stars.                       |

| Ambidexterity                                                                                                                            |
| ---------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Needing to operate something with both hands equally skillfully, such as when dual-wielding weapons, or playing the piano. |
| _Passively:_ Lets you ignore any penalties that might otherwise apply from using an 'off-hand', since you don't have one.                |

| [animals]                                                                                                                                                                                                           |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Interacting or working with an animal of a given type.                                                                                                                                                |
| _Passively:_ Ensures that this animal will never indirectly harm you, or that they will never accidentally throw you from your saddle.                                                                              |
| _**Examples:** Cats, Dogs, Wolves, Bears, Cows, Bats, Frogs, Songbirds, Birds of Prey, Fish, Sharks, Lizards, Snakes, Shellfish, Insects, Spiders, Jellyfish, etc._                                                 |
| _**(steed) Examples:** Horses, Mules, Camels, Elephants, etc, or specially-bred Dogs, Bears or Rhinos, or perhaps even more exotic things like Great Birds, Dinosaurs, or Giant Arthropods if your setting allows._ |
| #que Try to favor natural / linguistic groupings of animals, rather than strict scientific terms                                                                                                                    |

| Appraisal                                                                                                                                                              |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Judging the material worth of an object or artefact, or identifying what it does or what it can be used for, based on technical insights and experience. |

| Architecture                                                                                                      |
| ----------------------------------------------------------------------------------------------------------------- |
| **Use when:** Designing buildings, roads and bridges, public spaces, art installations, and other civic projects. |

| [armor]                                                                                                                 |
| ----------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Wearing, making, or repairing armor of the given type.                                                    |
| _Passively:_ Ensures that the armor does not impede your mobility, chafe, or otherwise induce any unexpected penalties. |
| _**Examples:** Unarmored, Light, Natural, Medium, Heavy, Superheavy, Mechanical._                                       |

| Athletics                                                                                                                                                        |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Physically exercising for recreation or survival, or when participating in a general sport, especially for cardiovascular activities like running. |
| #que Make up and use a [sport] specialty for specific activities                                                                                                 |

| Baking                                                               |
| -------------------------------------------------------------------- |
| **Use when:** Making breads, cakes, pastries, biscuits, pies, etc.   |
| _Passively:_ Ensures that you always remember to pre-heat your oven. |

| Barter                                                                                                                       |
| ---------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Exchanging goods and services of roughly equal value, and debating what that value is.                         |
| _Passively:_ You can always round in your favor when buying or selling (if you're tracking individual coins and currencies). |

| [biomes]                                                                                                                                                                                                                                                     |
| ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Use when:** Navigating, exploring, or otherwise surviving in the given biome type.                                                                                                                                                                         |
| _**Examples:** Rocky Mountain, Steppe / Prairie, Temperate Forests, Tropical Jungle, Hot Desert, Dry Savanna, Farmland, Swamp / Wetland, Taiga (cold desert), Tundra (arctic), Underground, Island, Lake, Ocean, Underwater, Hellscapes, Alien Climes, etc._ |

| Brewing                                                          |
| ---------------------------------------------------------------- |
| **Use when:** Making alcohol, potions, poisons, etc.             |
| _Passively:_ Ensures that you never get high on your own supply. |

| Bushcraft                                                                                                                                                  |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Use when:** Finding food, water, and shelter in the wilderness; specifically, when using skills related to making and using tools that help you survive. |
| _Passively:_ You can always find enough dry material to build a fire.                                                                                      |
